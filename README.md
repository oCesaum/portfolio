# Portfolio

Projeto de portfólio desenvolvido com Next.js, TypeScript e Tailwind CSS.

## Desenvolvimento

```bash
npm install
npm run dev
```

Aplicação local:
- `http://localhost:3000`

## Estrutura principal

- `src/app/`: rotas e entrypoints da aplicação
- `src/components/`: componentes de interface
- `public/`: assets estáticos
- `docs/`: documentação canônica do projeto

## Documentação

O ponto de entrada oficial da documentação está em:

- `docs/00-hub/documentacao-hub.md`

## Comandos úteis

```bash
npm run dev
npm run build
npm run start
npm run lint
```
