# Hub de documentação

Este é o ponto de entrada canônico da documentação do projeto.

## Visão geral

- Projeto: portfólio web em Next.js com Tailwind CSS.
- Objetivo atual: manter uma base documental e operacional compatível com Codex e Cursor.
- Fonte canônica: pasta `docs/`.

## Regras operacionais

- Antes de criar novos documentos, verifique se já existe um arquivo ativo que possa ser atualizado.
- Documentos novos ou atualizados devem manter frontmatter com metadados mínimos.
- Materiais históricos ou substituídos não devem ser usados como referência principal sem validar o status.
- `AGENTS.md` é configuração local do workspace e não deve ser versionado por padrão.
