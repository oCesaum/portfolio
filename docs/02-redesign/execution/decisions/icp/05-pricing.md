# ICP Pricing — quanto cada perfil paga e como ele paga

## A — Recruiter BR (CLT/PJ, full-stack mid)
- **CLT** mid: R$ 7k–13k · top-25% R$ 13k–17k. Benefícios ~+25%.
- **PJ** mid: R$ 12k–22k · top-25% R$ 22k–28k.
- **BRL/mês:** R$ 9k–22k (mediana ~R$ 14k).
- **Cobrança:** mensal fixa, hora não se aplica.
- **Floor:** R$ 6k · **Ceiling:** R$ 28k.
- **Per-project:** ❌ incompatível (vínculo). **Risco:** baixo CLT / médio PJ (NF atrasa 15-30d).
- **Score econ.:** 7/10 — teto bom, consome 100% do tempo.

## B — Internacional remoto (LATAM contractor USD)
- Mid: **USD 25–55/h** · top-25% **USD 55–85/h** (Toptal, Crossover, YC).
- 140h/mês: USD 3.500–7.700 → **R$ 18k–40k** · top-25% R$ 40k–62k.
- **Cobrança:** hora rastreada (Deel, Upwork, Hubstaff); retainer só após trust.
- **Floor:** USD 18/h (~R$ 13k) Upwork commodity. **Ceiling:** USD 100/h (~R$ 73k) niche AI/Stripe.
- **Per-project:** ⚠️ parcial — fixed-bid em SOW <USD 8k ok, resto hourly. **Risco:** baixo via escrow.
- **Score econ.:** 9/10 — melhor R$/h, mas exige inglês B2+ e rampa 2–4 meses.

## C — Founder BR (landing/MVP)
- **Landing premium:** R$ 3,5k–8k · top-25% R$ 8k–15k.
- **MVP enxuto** (auth + 2-3 telas + Stripe/PIX): R$ 12k–35k · top-25% R$ 35k–70k.
- **BRL/mês** (1 landing + 0,5 MVP entregues): **R$ 10k–25k** realista solo noturno.
- **Cobrança:** ✅ fixed-price por escopo é o padrão — founder odeia hora aberta.
- **Floor:** R$ 2,5k (bootstrap). **Ceiling:** R$ 70k (seed).
- **Per-project:** ✅ ideal — match com "cobro pelo problema". **Risco:** **alto** (sumiço, scope-creep). Mitigar: 50% upfront, marcos.
- **Score econ.:** 8/10 — alinhado a §6, mas volatilidade pede pipeline 3x.

## D — Agência BR (subcontratação)
- **Day-rate** mid: R$ 800–1.400 · top-25% R$ 1.400–2.000.
- **Project-rate:** 60–70% do que agência cobra → R$ 8k–25k típico.
- 12–15 dias/mês: **R$ 10k–21k** · top-25% R$ 21k–30k.
- **Cobrança:** day-rate é default; closed-scope só se César forçar.
- **Floor:** R$ 500/dia (interior). **Ceiling:** R$ 2.500/dia (SP/RJ premium).
- **Per-project:** ⚠️ negociável. **Risco:** médio — NF 30/45/60d, default raro, atraso constante.
- **Score econ.:** 7/10 — fluxo estável, margem comprimida, vira "mão".

## Compatibilidade com §6 ("cobro pelo problema")

| ICP | Per-project? | Hora forçada? |
|---|---|---|
| A Recruiter | ❌ | n/a (é salário) |
| B Internacional | ⚠️ parcial | sim, após SOW inicial |
| C Founder | ✅ **nativo** | raríssimo |
| D Agência | ⚠️ negociável | day-rate é default |

## Ranking Y1 realista (César solo, noturno, BR)

1. **C Founder BR** — R$ 12k–22k/mês realista · 100% aderente ao princípio · canal direto pro posicionamento "resolvo problema, não vendo hora".
2. **B Internacional** — R$ 18k–35k/mês teto maior, mas Y1 fica em R$ 8k–15k (rampa de reputação Upwork/Toptal).
3. **D Agência BR** — R$ 10k–18k/mês estável Y1 · bom como base enquanto C amadurece.
4. **A Recruiter BR** — R$ 9k–14k/mês CLT/PJ · only-fallback: mata tempo de construir asset próprio e viola §6.

**Recomendação:** C como foco · D como cushion · B como upside Y2 · A só se runway acabar.
