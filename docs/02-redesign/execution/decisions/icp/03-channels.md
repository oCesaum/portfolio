# ICP Channel Map (César's current state)

State assumed: LinkedIn active, GitHub public, portfolio site live, WhatsApp operational. No Toptal/Upwork profiles, no agency partnerships, no paid ads, no English-language content cadence.

## A — Recruiter BR
1. **Discovery channels:** LinkedIn search/Recruiter, Gupy/Kenoby ATS pools, indications via dev Slack/Discord BR (Codecademy BR, He4rt, Frontend BR).
2. **Evaluation artifacts:** LinkedIn headline + experience block, GitHub contribution graph, portfolio "About" page, salary/location signals.
3. **César can appear in:** LinkedIn (yes), GitHub (yes), portfolio (yes). Cannot appear in private ATS pools without applying first.
4. **Setup cost:** R$ 0; ~4h to optimize LinkedIn headline/keywords + pin GitHub repos.

## B — International remote
1. **Discovery channels:** LinkedIn (English profile + Open to Work remote), Wellfound/Hacker News "Who is hiring", curated marketplaces (Toptal, Arc, Lemon.io, Gun.io).
2. **Evaluation artifacts:** English portfolio with case studies + metrics, GitHub READMEs, Stack Overflow/Dev.to writing, Loom intro, time-zone overlap statement.
3. **César can appear in:** LinkedIn EN (partial — needs translation), Wellfound (free, fast), HN "Who wants to be hired" (free monthly thread). Toptal/Arc gated by screening (10–25h prep + 2–4 weeks).
4. **Setup cost:** R$ 0 cash; ~12h to translate site, write 2 case studies in EN, create Wellfound profile. Toptal funnel adds ~30h spread over a month.

## C — Founder BR
1. **Discovery channels:** Twitter/X BR tech, indie/founder communities (Cursos.dev, Codante Discord, Maker BR Telegram), referrals from previous clients/peers.
2. **Evaluation artifacts:** Live shipped product (the portfolio itself counts), Twitter build-in-public threads, case studies showing ROI/time-to-ship, WhatsApp responsiveness.
3. **César can appear in:** Twitter/X (yes — needs cadence), portfolio case studies (yes), WhatsApp CTA (yes). Founder communities require active participation, not just lurking.
4. **Setup cost:** R$ 0; ~6h to set up Twitter bio + pinned thread; ongoing 2h/week content cost.

## D — Agência BR
1. **Discovery channels:** Direct referral from agency owners César already worked with, LinkedIn outbound (agency CTO/Tech Lead targeting), agency-focused Slacks (RD Station partners, Resultados Digitais ecosystem).
2. **Evaluation artifacts:** Portfolio with white-label/agency-friendly framing, case studies with deadlines met, NDA-friendly references, hourly/project rate sheet.
3. **César can appear in:** LinkedIn outbound (yes), portfolio (yes — needs an "agencies" angle). No existing agency network to tap warm referrals.
4. **Setup cost:** R$ 0 cash; ~15h to build outbound list (50 agencies) + write 3 personalized templates + add agency landing section.
