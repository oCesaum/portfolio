# Time-to-Paid by ICP

Realistic estimates assume César ships portfolio v2 publicly, then runs ICP-specific outbound/inbound for 2 hrs/day. "Days" = calendar days from launch (T+0) to first cleared BRL.

## A. Recruiter BR (CLT / PJ via recruiter)

- **Sales cycle:** 35–55 days. Standard funnel: apply/be-sourced → screening (3–7d) → tech challenge (5–10d) → 2–3 panels (7–14d) → offer (3–7d) → CLT/PJ paperwork (5–10d) → 30-day first paycheck.
- **Lead-to-close ratio:** ~1 in 25–40 applications for mid/senior full-stack BR.
- **Onboarding:** 30 days until first salary lands (CLT folha) or 15–30 days for PJ NF.
- **First BRL realistic:** **70–95 days.** Portfolio v2 helps pass tech screen but doesn't compress HR/legal latency.
- **Speed score:** **4/10** — predictable but slow; payout only after onboarding cycle.

## B. International remote (Toptal / Upwork / direct)

- **Sales cycle:**
  - *Toptal:* 14–28d screening (English call → live coding → test project) before any client. Then 7–21d to first client match. Total 21–49d to first paid hour.
  - *Upwork:* faster — first proposal to first contract ~10–25d if portfolio is strong; new accounts fight Connects + no-history penalty.
- **Lead-to-close ratio:** Toptal screening ~1/1 (binary pass). Upwork ~1 contract per 30–60 proposals as a new seller.
- **Onboarding:** Upwork escrow releases 5–14 days after milestone; Toptal weekly invoicing; Payoneer/Wise to BR ~2–4 business days.
- **First BRL realistic:** **35–60 days** (Upwork milestone path) / **45–75 days** (Toptal). Portfolio v2 is a strong unlock here — international clients buy on visual proof.
- **Speed score:** **6/10** — moderate speed, low ceiling on first ticket but high LTV.

## C. Founder BR (WhatsApp inbound, indicação)

- **Sales cycle:** WhatsApp ping → discovery call (1–3d) → proposal (2–5d) → signed contract + 50% upfront (3–7d). Total **7–15 days** when warm.
- **Lead-to-close ratio:** ~1 in 3–5 qualified inbound chats for an indicação-driven funnel; ~1 in 15–25 for cold.
- **Onboarding:** PIX 50% upfront clears same day. First BRL = day of signature.
- **First BRL realistic:** **10–25 days** if César activates network on launch day (anuncia portfolio v2 no LinkedIn/WhatsApp + pede 5 indicações). **30–45d** if relying on organic inbound.
- **Speed score:** **9/10** — fastest path; PIX kills the payment latency tax.

## D. Agência BR (cold email / parceria)

- **Sales cycle:** Cold email/DM → reply (3–10d) → call (5–10d) → trial paid project (10–20d) → first NF paid 15–30d net. Total 35–70d.
- **Lead-to-close ratio:** ~1 trial per 40–80 cold touches; ~1 in 8–12 if warm-intro via LinkedIn 2nd-degree.
- **Onboarding:** Most agências pay 15/30 ddl after NF, not upfront. Cash hits bank 45–75d after first contact.
- **First BRL realistic:** **50–80 days.** Recurring after the first project but slow ignition.
- **Speed score:** **5/10** — solid LTV/recurrence, brutal time-to-first-real.

## Summary table

## Decision

**Fastest ICP to first revenue: C. Founder BR via WhatsApp inbound + indicações** — 10–25 days, PIX upfront, portfolio v2 closes the trust gap that warm intros need.
